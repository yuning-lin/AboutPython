## 程式重構範例
* [單元測試（unit test）](https://github.com/yuning-lin/PythonTips/blob/main/Refactoring/unit_test.ipynb)
  1. 寫好單元測試，對原程式進行測試
  2. 重構後再行測試，確保沒有破壞程式
  3. PLUS：配合版本控制，可以避免錯誤無法回復
